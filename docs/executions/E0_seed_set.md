# Execution log: E0 seed set

- **execution_id**: E0
- **execution_type**: manually curated seed set

## Purpose
Create a controlled, high-precision, auditable initial corpus for later snowballing executions.

## Input sources
- Prior domain expertise and known key references.
- Transparent manual curation notes.
- Allowed discovery interfaces used only for targeted verification (not broad harvesting).

## Expected output
- A first seed candidate table with traceable inclusion rationale.
- Core/contextual/candidate/rejected seed statuses documented.

## Screening rule
Include only records with explicit and defensible relevance to criminal infiltration in the legal economy (core) or essential contextual support (contextual).

## Saturation rule
E0 does not assess saturation. E0 establishes the baseline corpus, therefore candidate novelty rate and eligible increment rate are not substantively interpretable for saturation.

## Limitations
- Potential omission of relevant papers due to precision-first strategy.
- Dependence on manual judgement at initialization stage.
