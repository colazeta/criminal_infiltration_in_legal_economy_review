# Literature Review Protocol

## Aim
Build a systematic, execution-based snowballing literature review on criminal infiltration in the legal economy.

## Research questions
1. Which mechanisms are used to infiltrate legal firms, sectors, and markets?
2. Which sectors, governance arrangements, and ownership structures are most exposed?
3. Which empirical methods and data sources are used to measure/describe infiltration?
4. Which outcomes are associated with criminal infiltration in legal economic activity?

## Scope
- Topic: criminal infiltration/control/influence/penetration in the legal economy.
- Unit of analysis: paper-level evidence and discovery-event-level provenance.
- Output: auditable registry, screening decisions, thematic coding, saturation metrics.

## Inclusion/exclusion logic
- Include papers that substantially address infiltration mechanisms, channels, or impacts in legal economic contexts.
- Exclude papers with only passing mention, unrelated crime topics, or no legal-economy link.
- Handle ambiguity with a staged decision (`maybe_full_text_needed`) before final decision.

## Unit distinction
- **Paper-level unit**: canonical record for each unique scholarly item.
- **Discovery event**: each retrieval occurrence per execution/feed/query/source-paper context.

## Execution-based workflow
1. Seed retrieval from approved feeds.
2. Backward and forward snowballing from screened source papers.
3. Deduplication to canonical paper registry.
4. Stage-1 screening (title/abstract), stage-2 screening (full text if needed).
5. Eligibility decision + exclusion code logging.
6. Thematic coding and execution-level saturation update.
